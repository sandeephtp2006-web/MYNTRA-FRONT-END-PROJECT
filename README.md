# MYNTRA-FRONT-END-PROJECT
A static Myntra-style homepage built using HTML and CSS. Includes navigation menu, search bar, category sections, promotional banners, and product grids. Designed with absolute positioning for layout and styled components to replicate the look of an e-commerce landing page.
MYNTRA Homepage Clone
